Active GUI (supported)
======================
radar_dashboard.py  ==> Board bring-up dashboard (FT2232H reader, real-time R-D
                        heatmap, CFAR overlay, waterfall, host commands, HDF5
                        recording)
radar_protocol.py   ==> Protocol layer (packet parsing, command building,
                        FT2232H connection, data recorder, acquisition thread)
smoke_test.py       ==> Board bring-up smoke test host script (triggers FPGA
                        self-test via opcode 0x30)

Historical GUIs (archived, unsupported)
=======================================
The GUI_V1 … GUI_V6 prototypes have been moved to  archive/  and are kept only
for reference. See  archive/README.md  for the change log and a warning about
stale opcode values.
